### 1. Rodando Projeto Localmente

##### 1.1. Tenha Instalado

* [Java JDK 11]
* [Maven]
* [Docker]

##### 1.2. Passo-a-Passo

1. Clone o projeto;
2. Através do terminal entre na pasta do projeto;
3. Compile o Maven;
4. Abra sua IDE favorita, importe o projeto e o execute normalmente.

```sh
1. git clone <url_do_projeto>
2. cd <nome_da_pasta>
3. mvn package -U
```
